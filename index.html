🌐 Enable Multi-Device Sync

ℹ️
Firebase sync requires the app to be served over HTTPS. The easiest way is GitHub Pages — free, instant, no server needed.
1
Create a GitHub account & new repository
Go to github.com → Sign up (free) → Click + New repository → Name it admission-test → Set to Public → Create.
2
Upload this HTML file
In your new repo → Click Add file → Upload files → Drag the admission-test-app-v2.html file → Rename it to index.html → Commit changes.
3
Enable GitHub Pages
In your repo → Settings → Pages → Source: Deploy from branch → Branch: main → / (root) → Save. Your app will be live at https://yourusername.github.io/admission-test/ in ~60 seconds.
4
Enable Firebase Firestore
Go to console.firebase.google.com → admission-test-v2 → Firestore Database → Rules → Replace with:
allow read, write: if true; → Publish. (Switch to auth-based rules later for production.)
5
Open your GitHub Pages URL & connect Firebase
Open https://yourusername.github.io/admission-test/ → Admin login → Settings → Configure Firebase → Click Connect Firebase. Done! Every device that opens this URL shares the same data.
✅
Your Firebase project is already configured in the app — admission-test-v2. Once hosted on GitHub Pages, just click Connect and it will work immediately.

Set Firestore rules to allow access
